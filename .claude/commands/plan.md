# /plan — 実装前計画提示

**コードは一切変更せず**、以下の調査と計画だけを実行してください。

1. 関連ファイルを `grep` / `find` で最小限に調査する
2. 現在のアーキテクチャ上で影響するファイルを列挙する
3. 下記形式で実装計画を日本語で提示する:

```
## 実装計画: [ARGUMENTS]

### 影響範囲
- 変更ファイル: ...
- 依存先モジュール: ...

### 実装ステップ
1. ...
2. ...

### リスク / 注意点
- ...

### 完了条件
- [ ] cargo clippy クリーン
- [ ] npm run typecheck クリーン
- [ ] テストパス
```

4. 「この計画で進めますか？」と我に確認する

**対象**: $ARGUMENTS
